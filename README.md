# 📚 lectures

이 저장소는 Inflearn, YouTube, FastCampus, 공식 문서 등의 플랫폼에서 수강한 강의들의 학습 내용을 정리한 공간입니다. 각 플랫폼별 디렉토리 아래 강의별로 내용을 분류하여 정리하였습니다.

```plaintext
lectures/
├── inflearn/ # Inflearn 강의 정리
├── fascampus/ # FastCampus 강의 정리
├── youtube/ # YouTube 영상 강의 정리
```
