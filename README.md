# Spring Multi Module 학습 - 패스트캠퍼스, 10 개 프로젝트로 완성하는 백엔드 개발

[패스트캠퍼스](https://fastcampus.co.kr)의 [10개 프로젝트로 완성하는 백엔드 개발(Java/Spring) 초격차 패키지 Online)[https://fastcampus.co.kr/dev_online_befinal) 강의 중

## 개발 환경

* Java 17
* Gradle 8.3
* Spring Boot 3.1.5
