## 개발 환경

* Intellij IDEA Ultimate 2023.2.3
* Java 17
* Gradle 8.3
* Spring Boot 3.1.5

## 기술 세부 스펙

* Spring Web
* Lombok
* Logback


## Appender 종류

* ConsoleAppender: 콘솔에 log 출력
* FileAppender: 파일 단위로 log 저장
* RollingFileAppender: log 를 여러 파일로 나누어 저장
* SMTPAppender: log를 메일로 전송하여 기록
* DBAppender: log를 DB에 저장
